# 截图

跑起来 demo 后，浏览器全屏截图保存到这里：

- `dashboard.png` —— 完整仪表盘截图（首屏即可，1600×1000 左右最佳）
- `mobile.png`（可选）—— 手机端截图，证明响应式布局工作

主 README 和 demo README 里的截图链接会自动指过来。

## 怎么截图

1. 跑起来 demo：`cd demo && python app.py`
2. 浏览器打开 http://localhost:5000，等所有图都加载完
3. 截全屏：
   - **Mac**：`Cmd + Shift + 4`，按空格再点击窗口
   - **Windows**：`Win + Shift + S`
   - **Linux**：用 GNOME Screenshot 或 Spectacle
4. 保存为 PNG，放到这个目录
