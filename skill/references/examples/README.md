# 参考案例

四个典型场景的精简案例。用户的需求跟哪个最像就读哪个，**不用全读**。

| 文件 | 适合的数据形态 |
|---|---|
| `temperature.md` | 时间序列 + 多地点 + 数值随季节波动（气温、销量、流量、能耗） |
| `weather.md` | 实时数据 + 严重等级 + 地理分布（天气预警、IoT 传感器、事件日志） |
| `lifespan.md` | 静态统计 + 多维分类（人口、收入、教育数据，按地区×性别×年龄段拆分） |
| `project.md` | 运营跟踪 + 状态 + 截止日期（Jira、销售管线、客服工单、OKR） |

`temperature.md` 对应仓库根目录的 `demo/` 项目，可以直接对照看完整代码实现。
